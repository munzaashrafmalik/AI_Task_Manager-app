# AI Task Manager Pro 🚀
# URL: https://ai-task-manager-app-munza.netlify.app/
A professional, feature-rich task management application with AI-powered suggestions built with pure HTML, CSS, and JavaScript.

## ✨ Features

### Core Features
- 📝 **Create/Edit/Delete Tasks** - Full CRUD operations
- ✅ **Task Completion** - Mark tasks as complete with animations
- 📅 **Due Dates & Times** - Schedule tasks with specific dates and times
- 🔥 **Priority Levels** - High, Medium, Low priority tagging
- 🏷️ **Categories** - Organize tasks by customizable categories
- 📊 **Dashboard Analytics** - Visual progress tracking and statistics
- 🔍 **Search & Filter** - Find tasks quickly
- 🔄 **Sort Tasks** - By date, priority, name, or category
- 📱 **Fully Responsive** - Works on all devices

### Advanced Features
- 🧠 **AI Suggestions** - Smart task recommendations
- 🤖 **AI Auto-Categorization** - Automatically categorize tasks
- 📋 **Subtasks** - Break down tasks into smaller steps
- 🔁 **Recurring Tasks** - Set repeating tasks
- 🌓 **Dark/Light Theme** - Toggle between themes
- 💾 **Local Storage** - Data persists automatically
- 📈 **Weekly Progress** - Track your productivity
- ⏰ **Overdue Alerts** - Never miss a deadline

### Views
- **Dashboard** - Overview with stats and AI suggestions
- **All Tasks** - Complete task list with filters
- **Today** - Tasks scheduled for today
- **Upcoming** - Future tasks
- **Completed** - Task history

## 🚀 Quick Start

### Step 1: Get OpenAI API Key (Optional)
For AI features:
1. Visit [OpenAI Platform](https://platform.openai.com)
2. Sign up and get your API key
3. Copy the key (starts with `sk-...`)

### Step 2: Open the App
Simply double-click `index.html` in your browser!

### Step 3: Configure AI Settings (Optional)
1. Click "AI Settings" in the sidebar
2. Enter your OpenAI API key
3. Enable/disable AI features
4. Click Save

## 🎯 How to Use

### Creating Tasks
1. Click "Add Task" button
2. Enter task title and description
3. Select category and priority
4. Set due date and time
5. Add subtasks (optional)
6. Click Save

### Managing Tasks
- **Complete:** Click the circle checkbox
- **Edit:** Click the edit icon
- **View:** Click on the task card
- **Delete:** Click the trash icon

### Using Filters
- **Categories:** Click category in sidebar
- **Priority:** Click priority filter buttons
- **Search:** Type in search bar

### Views Navigation
- **Dashboard:** Overview and stats
- **All Tasks:** Complete list
- **Today:** Today's tasks
- **Upcoming:** Future tasks
- **Completed:** Done tasks

## 🎨 Customization

### Add Custom Categories
1. Click "Add Category" in sidebar
2. Enter category name
3. Pick a color
4. Click Add

### Change Theme
Click the moon/sun icon in sidebar to toggle themes.

## 🛠️ Troubleshooting

**Tasks not saving?**
- Check if browser allows Local Storage
- Try a different browser

**AI features not working?**
- Verify API key is correct
- Check internet connection
- API key might need activation time

**Theme not changing?**
- Clear browser cache
- Refresh the page
